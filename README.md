# 地址坐标计算 (feishu-geo-calculator)

飞书多维表格字段捷径 — 地理编码/逆地理编码/距离计算/半径检索。

## 功能

- **地理编码**：地址文本 → 经纬度坐标
- **逆地理编码**：经纬度坐标 → 详细地址
- **距离计算**：两点之间的距离（Haversine公式，纯本地计算）
- **半径检索**：查找指定坐标半径范围内的点

## 服务商

- **Nominatim**（免费，无需API Key，基于OpenStreetMap数据）
- **高德地图**（需API Key，国内地址精度高）
- **百度地图**（需API Key，国内地址精度高）

## 配置

1. 新建字段 → 探索字段捷径 → 选择「地址坐标计算」
2. 选择操作模式和服务商
3. 配置对应的输入字段
4. 运行即可

## 注意

- 距离计算和半径检索无需网络请求（纯本地计算）
- Nominatim免费版有1请求/秒的频率限制
- 高德/百度地图需要从对应开放平台申请API Key
